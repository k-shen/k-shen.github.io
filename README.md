# k-shen.github.io

Personal portfolio website hosted on GitHub Pages at [kaiwenshen.com](https://kaiwenshen.com).

Built with HTML, CSS, Bootstrap 4, and jQuery. Page sections are split into individual files under `sections/` and loaded dynamically via `fetch`.

## Local Development

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080` in your browser.
